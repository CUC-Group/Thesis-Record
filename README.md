# 节点重要性组文献记录

## 基本了解

本Repo主要用于记录节点重要性组每周组会分享文献内容，目前主要包括复杂网络和重要节点识别方向的发表论文。同时包含了本组成员对所阅读文献的总结与分析笔记，方便后续查阅和新生跟进。

## 上传格式

一般应包括：

1. 原文献的pdf文件
2. 根据文献所作的阅读笔记，根据组内模板，应为word文件
3. 一个简略介绍文献内容的README文件
4. 一个在README中显示期刊分区的sci.png/jpg图片文件

## 使用方法

1. 根据排序和commit查看文献内容，寻找需要的文献
2. 在网页端使用Ctrl + F寻找关键词
